## Инструкция по работе    

### Где брать задания

1. Репозитории https://github.com/Finuniversity/ADSPython.git    
2. Папка Task    
3. Файл Task.md    

### Как сдавать задания    

В своем репозитории создаете папку с именим задания (Пример 1-Task, 2-Task итд) и туда размещаете файлы
